# Projeto IoT com Raspberry Pi Pico W e MQTT

## Objetivo
Projeto desenvolvido utilizando Raspberry Pi Pico W para demonstrar comunicação IoT através do protocolo MQTT.

## Funcionalidades
- Conexão Wi-Fi
- Conexão com Broker MQTT
- Leitura da temperatura interna da Pico W
- Publicação da temperatura em tópico MQTT
- Controle remoto do LED onboard através de mensagens MQTT

## Tópicos MQTT
Temperatura:/Mateus/240025564/temperatura
LED:/Mateus/240025564/led


## Tecnologias
- Raspberry Pi Pico W
- C/C++
- Pico SDK
- MQTT
- LWIP
- Wi-Fi
